Class loo nannu baaga artham chesukunevaaru evarante adhi snehith.
Endhukante nenu thanaku cheppadam complete kakamundhe thanaku ventane artham ayipothundhi, ante thanaku listening skills entha perfect ga vundho think cheyandi.
Thanaku andharitho friendship cheyalane vuntundhi.
Nijaniki thanaku communication skills chaala ekkuve but endhukani thanu aa skills ni implement cheyado thelidhu, so thanu correct ga thanu thana skills ni use chesukunte inka better postition lo vuntadani naa feeling.
Thanaku emaina personal vishayalu chebuthe personal vi personal gaane vunchuthaadu and cheppindhi kuda baaga gurthupettukuntaadu.

Good habits:
-> Thanadhi friendly nature.
-> Evaremaina chebuthunte concentration tho vintaadu.
-> Helping nature kuda ekkuve.
-> Emaina cheppagane ventane respond avuthaadu.

Bad habits:
-> Thanu endhuko class loo silent ga anipisthaadu so ala kakunda konchem andharitho friendly ga move ayithe inka better ani naa feeling.
