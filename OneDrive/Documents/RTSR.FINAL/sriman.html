Nickname: "Mute Maestro"

Asalu evaraina naa dhaggariki vachi sriman class ki vachaada ani aduguthe nenu kachithanga confuse avuthaanu, asalu thanu vachaada raledha ani, ante maa idhari madhya communication entha thakkuvo arthamchesukondi.
Memu oka sari conversation start chesamante ika evening varaku konasaguthadhi, but okkasari conversation stop chesaamante ika konni months varaku distance konasaguthadhi.
Memu idharam same district nunchi vachina vallame so andhuke konchem maa idhari madhya conversation ekkuvasepe nadusthundhi.
Thanaku eppudaina edhaina work vunte mathram, thanu thvaraga complete cheyyalani try chesthaadu.

Hidden skill:

-> Nijaniki thanaku communication skills chaala ekkuve, but thanu aa skill ni endhuko ekkuvaga use chesekodo theliyadhu.

Good habits:

  • -> Andharitho friendly gaane move avuthaadu.
  • -> Active listening skills kuda ekkuve.
  • -> Evaritho godava padalani chudadu.

Bad habits:

  • -> Class loo ala silent ga kakunda konchem andaritho interact avuthe better ani naa feeling.
