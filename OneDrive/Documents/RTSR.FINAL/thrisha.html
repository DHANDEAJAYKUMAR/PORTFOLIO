Nijaniki neeku class loo evaritho matladalani vundhi evaraithe vaaru neetho anthala matladaro ani nannu adigithe naaku first gurthukochedhi Thrisha. Endhukante asalu nenu thanatho matladalani chalasarlu anukunta but alanti situations hee raludhu. Nijaiki class loo thanaku vunnantha communication skills inkevvariki vundadhu naa feeling. Thanu seminar ichinappudalla nenu shock ayyevaanni thana communication skills ni chusi. so inka ilane thana skills ni correct ga use chesukunte velthe inka top position lo vuntundhani naa feeling. Thanu andharithonu free ga matlade ammayi. Good habits: -> Face-to-Face matlade person. -> Thanu friendship ki entha value isthundho studies ki kuda anthe value isthundhi. -> Thanaku helping nature kuda ekkuve. -> Eppudu ela behave cheyaalo thanaku baaga thelusu. -> Boys tho communicate chesetappudu thanu konchem limits lo vuntundhi. -> Thanu edhaina thappu chesthe aa thappu oppukuntundhi, okavela thanu ee thappu cheyyakapothe mathram thala vanchadhu. Bad habits: -> Thanu endhuko class loo silent ga anipisthundhi so ala kakunda konchem andharitho friendly ga move ayithe inka better ani naa feeling.
